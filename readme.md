# readme

Python client for dotpack ( Eliteu Programable led matrix bag )

# Install

```bash
# Python >= 3.6
pip install dotpack
```

# Usage

```python
from dotpack import DotPack
```

# Discover the dotpack

```bash
bleak-lescan
```
